Synthèse par table d'ondes

Une table d'ondes est un tableau de N échantillons représentant exactement une période d'un signal. Pour jouer une note à la fréquence f avec un taux d'échantillonnage SR, on avance dans la table à la vitesse N·f/SR échantillons par pas. Le timbre (contenu harmonique) est entièrement défini par la forme de la table ; la hauteur change sans altérer le timbre. On peut aussi interpoler entre deux tables pour faire évoluer le timbre.
